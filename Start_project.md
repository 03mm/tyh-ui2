# Start_project

## Start project

1. Fork [tyh-ui2](https://github.com/Tyh2001/tyh-ui2)

2. install

```shell
git clone git@github.com:Tyh2001/tyh-ui2.git

cd tyh-ui2

npm i

npm run dev
```
